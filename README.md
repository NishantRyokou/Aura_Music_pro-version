# Aura_Music_pro-version
Shipped major upgrades to Aura_Music_pro version on microsoft: (ms-windows-store://pdp/?productid=9NBLL4FLBVK6)

Core Build: Delivered as a single 80MB executable — one click to run on any desktop device.

Frameworks: Built with Electron + Vite, leveraging Kotlin + Java for cross-platform compatibility.

Sidebar Clock: Live clock displayed at the top-left next to Aura heading.

Notepad: Inline rename, export, cursor tracking, and a 1M word limit per pad.

Tasks: Infinite nesting, progress graphs, and IndexedDB v3 schema.

Calendar: Sidebar mini-calendar with event dots + export.

Alarm Timer: Powered by Web Audio API with custom labels and sounds.

Player: Loop modes, lyrics fetch, grayscale cover art, Volume Boost (+6dB bus), Ultra Boost button, and enhanced full-capacity Hz cycle for richer desktop sound.

Library: Inline rename, duplicate prevention, alphabetical sort, sanitized uploads.

Search & Settings: Real-time filters, streamlined search bar, 100+ font styles, and text size scaling up to 2×.

Optimizations: Isolated clock component, memory leak fixes, type safety, and dead code cleanup.

Experience Upgrades: Loop button, ultra boost, alphabetically sorted library, duplicate removal, and enhanced search bar for best usability.
